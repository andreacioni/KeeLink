How To

In order to send password to this page you first need to follow 3 simple steps:

  1. Download and install needed app on your phone/tablet
    • Keepass2Android Password Manager (online or offline version)
    • KeeLink Plug-In for Keepass2Android(here)
  2. Open Keepass2Android and click on KeeLink button 'Send' in the password entry context menu
  3. On the opened window you can scan the QR code on this page and wait for password to be sent.
  4. Interested on details? Just go ahead!

How it works

KeeLink has a very simple and strong architecture, it is composed mainly of two parts:

  • The Android one
    • KeeLink Android application acts as a bridge from Keepass2Android to this web page. Every password is retrieved from the Keepass Database. Once the password is received, the application waits for the user to scan QR code with its device camera. QR Code contains a simple URL as ksid://yourpersonalanduniquesessionid, so when the QR code is correctly parsed, the password is sent to a database and associated with 'yourpersonalanduniquesessionid'. No other information are sent over network, that is important for you because it is essential for me to mantain your password as safe as possible. More security for your password was added by the HTTPS/SSL protocol that is used for real transmit from Android to World Wide Web.
  • The PHP one
    • KeeLink PHP application acts only as a simple server that build a custom session ID, random generated, only for you. It is converted to QR Code and dispayed at the top of the page. Once QR is received your page starts querying database and listening for an upcoming password.
