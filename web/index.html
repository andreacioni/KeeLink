How To

In order to send password to this page you first need to follow 3 simple step:

  1. Download and install needed app on your phone/tablet
    • Keepass2Android Password Manager (online or offline version)
    • KeeLink Plug-In for Keepass2Android(here)
  2. Open Keepass2Android and click on the password entry context menu as shown
  3. On the opened window you can frame the QR code on this page and wait for password to be sent.
  4. Interested on details? Just go ahead!

How it works

KeeLink has a very simple and strong architecture, it is composed mainly of two parts:

    The Android one

  • KeeLink Android application acts as a bridge from Keepass2Android and this web page. Every password are retrieved from the Keepass Database. Once the password was received the application wait for the user to scan the QR code with its device camera. QR Code contains a simple URL as ksid://yourpersonalanduniquesid, so when the QR code is correctly parsed the password was sent to a database and associated to 'yourpersonalanduniquesid'. No other information are sent over network, that is important for you because it is essential for our to mantain your password as safe as possible. More security for your password was added by the HTTPS/SSL protocol that is used for real transmit from Android to World Wide Web.
