How To

In order to send password to this page you first need to follow 3 simple step:

  1. Download and install needed app on your phone/table
    • Keepass2Android Password Manager (online or offline version)
    • KeeLink Plug-In for Keepass2Android(here)
  2. Open Keepass2Android and click on the password entry context menu as shown
  3. On the opened window you can frame the QR code on this page and wait for password to be sent.
  4. Interested on details? Just go ahead!
